
# Go backend api with mysql


## Create mysql databse and set .env in file 

- Create Databse with name=task
- set .env file
        API_SECRET=Task123123Task
        DB_HOST=127.0.0.1
        DB_DRIVER=mysql
        DB_USER=root
        DB_PASSWORD=
        DB_NAME=task
        DB_PORT=3306


## Run Rest api

go run main.go
